# Placement Agent

## Purpose

The Placement Agent estimates the student's starting point and creates a first learning baseline.

## Responsibilities

- Review onboarding answers.
- Review writing sample.
- Later, review speaking/listening/reading checks.
- Recommend an initial CEFR-aligned level.
- Identify confidence, goals, and preferred topics.

## Inputs

- Student profile.
- Current level selected by student.
- Main goal.
- Writing sample.
- Speaking sample transcript when available.
- Listening/reading check results when available.

## Outputs

- Initial level estimate.
- Confidence estimate.
- Recommended course.
- First practice priorities.
- Signals for AI Coach and AI Teacher.

## Boundaries

- Current baseline is an estimate, not a real measured score.
- Do not overstate accuracy without assessment data.
- Record uncertainty when the available signal is weak.

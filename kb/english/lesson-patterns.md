# Lesson Patterns

These patterns help the AI Teacher Agent generate consistent practice.

## Daily Check-In

Goal:

- Welcome the student.
- Mention one recent success.
- Mention one focus area.
- Start one short practice.

Pattern:

1. Greeting.
2. Progress note.
3. Today's focus.
4. First question or drill.

Example:

`Hi Lucas, good to see you. Yesterday you answered past-tense questions well. Today we will practice clear pronunciation of "thought" and "world". Say this sentence: "I thought the world was comfortable."`

## Guided Lesson

Pattern:

1. Objective.
2. Model.
3. Controlled practice.
4. Feedback.
5. Freer practice.
6. Recap.

## Pronunciation Drill

Pattern:

1. Target sound or word.
2. Teacher model.
3. Student attempt.
4. Feedback.
5. Repeat.
6. Sentence practice.
7. Short summary.

## Conversation Practice

Pattern:

1. Set topic.
2. Ask one open question.
3. Respond naturally.
4. Correct lightly.
5. Ask follow-up.
6. Save recurring errors after the conversation.

## Writing Practice

Pattern:

1. Give a prompt.
2. Student writes.
3. Identify strengths.
4. Correct priority issues.
5. Show improved version.
6. Student rewrites.

## Vocabulary Review

Pattern:

1. Retrieve due words.
2. Ask meaning or usage question.
3. Ask for original sentence.
4. Correct sentence.
5. Schedule next review.

# FluentPath English

AI-powered English learning platform with a personalized AI Teacher for every student.

## What is working today

- Student signup with full learning profile (level, goal, interests, address, motivation)
- AI placement test — 7 questions across grammar, vocabulary, reading, and listening, generated by Claude and evaluated server-side
- Placement baseline auto-runs after signup using the motivation text; students can retake at any time
- AI level suggestions — when Claude's placement differs from self-reported level, a suggestion is queued for teacher or admin review
- Teacher dashboard with assigned students, level suggestion approvals, and invite links
- Admin dashboard with student/teacher/course/plan management and school-wide level suggestion review
- Teacher invite links that auto-assign new students to the inviting teacher
- Account management (profile, address, password)
- Pronunciation attempt recording (metadata stored, audio upload coming)
- Local audio/video recording UI for in-person classes with consent gate
- PostgreSQL with in-memory fallback for local development
- Playwright smoke tests for all major flows

## Stack

| Layer | Technology |
|---|---|
| Frontend | Plain HTML + CSS + JavaScript, hash-based SPA |
| Backend | Node.js `http` (no framework), modular routes under `src/server/routes/` |
| Database | PostgreSQL via `pg`, in-memory fallback |
| AI | Anthropic Claude API (`claude-haiku-4-5-20251001`), prompt caching |
| Tests | Playwright end-to-end |
| Node | >= 24 |

## Setup

```bash
npm install
npx playwright install chromium
```

Copy the environment template:

```bash
cp .env.example .env
# Edit .env and set ANTHROPIC_API_KEY and DATABASE_URL
```

Run locally:

```bash
npm start
```

The site runs at `http://127.0.0.1:5173`. For Cloudflare Tunnel or Cloud Run, set `HOST=0.0.0.0`.

## Environment variables

| Variable | Required | Description |
|---|---|---|
| `ANTHROPIC_API_KEY` | Yes (for AI features) | Key from console.anthropic.com |
| `DATABASE_URL` | No | PostgreSQL connection string. If not set, uses in-memory demo data |
| `HOST` | No | Bind address (default `127.0.0.1`; use `0.0.0.0` for tunnels) |
| `PORT` | No | Port (default `5173`) |

## PostgreSQL setup

```bash
docker compose up -d postgres
node --env-file=.env scripts/migrate.js
node --env-file=.env scripts/seed.js
npm start
```

Or with exported variables:

```bash
export DATABASE_URL="postgres://USER:PASSWORD@localhost:5432/fluentpath"
npm run db:migrate
npm run db:seed
npm start
```

  agents/

  server/
    routes/

      pronunciation.js
